# Icons placeholder

Tauri ожидает файл `icon.ico` в этой папке для Windows-сборки.

В Sprint 0 не положен реальный icon — будет добавлен дизайнером позже.
Чтобы локально запустить `npm run tauri build`, положи любой `.ico` файл сюда.

Tauri также может сгенерировать набор иконок из одного PNG:
```
npx @tauri-apps/cli icon path/to/icon.png
```
